This program can be used to visualize points / objects / vectors like in math schoolbooks.

## Features
- Shapes like Rectangle, Circle, Cuboid, Tetrahedon
- 1D Functions, 2D Functions, Labeled Vectors
- Zoom-in /-out, Move view
